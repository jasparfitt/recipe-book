# Coook Website
Built with vite and react

## Development
To start install packages with `npm install` in the `/web` and `/shared` folder. Then link up the local shared code with `npm link ../shared/`. Then run `npm start` to start the webserver and visit it at localhost:3000

## Build
The website will build and deploy on every commit to main, but if you need to build manually you can run the [Github action](https://github.com/jasparfitt/recipe-book/actions/workflows/website.yml)
